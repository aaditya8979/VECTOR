# ⚡ Welcome to VECTOR

**Local. Offline. Verified. Zero hallucinations reach your code.**

VECTOR modifies your code using a local LLM running on your machine, then verifies every change through 5 deterministic layers before applying it.

### How it works:
1. **Builds a Code Property Graph** — maps every function and its relationships
2. **Compresses context** — extracts only what the LLM needs (≤2,500 tokens)
3. **Generates modification** — using your local model (Ollama, MLX, or GGUF)
4. **Verifies 5 layers** — syntax → symbols → types → tests → runtime
5. **Shows you a diff** — accept or revert with one click

### Supported Languages
Python · TypeScript · JavaScript · Go · Rust · C++ · C · TSX
